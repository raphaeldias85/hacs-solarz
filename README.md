# SolarZ – Home Assistant Integration

Integração não oficial do SolarZ para Home Assistant.

## Funcionalidades
- Autenticação via SolarZ
- Seleção de plantas/usinas
- Sensores:
  - Geração do dia (kWh)
  - Previsão do dia (kWh)
  - Status da usina

## Instalação via HACS
1. HACS → Integrações → Repositórios personalizados
2. URL: https://github.com/raphaeldias85/hacs-solarz
3. Categoria: Integração
4. Instale e reinicie o Home Assistant
5. Configurações → Dispositivos e Serviços → Adicionar integração → SolarZ

## Aviso
Este projeto não é afiliado ao SolarZ.
